Global variables:
An array to store the items of the grocery list

Functions:
Push an added item to the grocery array
Splice out the item from a given index of the grocery list
Swap a given index of the grocery list with the previous or next (update number in box too) with a temp var
Check array to see if indexOf the input is >0 (duplicate) in the grocery list and not add them
Spice out delimiter from input if indexOf >0 (present)
Check if inputted numbers are >0
Update quantity
Display
Empty the list with warning 

Interface: 
Logo on top left of the screen next to brand name
An ordered list based on the grocery list array displayed under the title
Show quantity number next to name if quantity>1
Radios to chose from adding or modifying list
When the add radio is selected, show button to add to grocery list.  Also show a text box for name and number box for quantity
When the modify radio is selected, show a button to delete, increase or decrease.  Also show a text box for position number
    As well as radio to chose from modifying position or quantity
Reset button

Flow: 
If the add radio button is selected, show a text box for the name of the item, a number box for quantity and a button to add
When the user clicks the add button, check if inputted number is >0
Check if delimiter is used in the item name.  If found, remove it
Then, check if the item exists on the list already.  
If it's not on the list already, concat the inputted name and the quantity, seperated by a delimiter (\)
Push the resulting string into the grocery list array
Display function will iterate through the grocery list and display the names in an ordered list.  Items with quantity >1 will also show quantity next to it

If the modify radio button is selected, show a number box for the position of the item in the grocery list.  Show buttons delete, increase and decrese.  Show radio to increase/decrease quantity or position
Check if inputted number is >0, and refer to the element at input-1
To delete, splice out the deleted item
To move up, store index in question in a temp var, index in the array = the element of prev index, then set the prev index = the temp var
Then, decrease the innerHTML of the box by 1
To move down, store index in question in a temp var, index in the array = the element of next index, then set the next index = the temp var
Then, increase the innerHTML of the box by 1
To increase quantity, increase the quantity part of the string of the index in question
To decrease quantity, decrease the quantity part of the string of the index in question
Update the display
